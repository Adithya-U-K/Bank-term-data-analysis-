# Bank-term-data-analysis
## Overview

This project analyzes the Bank Term Deposit Marketing Dataset, aiming to understand the factors influencing whether a customer subscribes to a term deposit.
The analysis includes exploratory data analysis (EDA), demographic insights, and visualizations to uncover key behavioral patterns that impact conversions.


## Tools & Technologies Used

Python 3
pandas – data loading and manipulation
numpy – numerical operations
matplotlib – plotting
seaborn – statistical visualizations

## Dataset 

It typically includes columns such as:
Age
Job
Marital status
Education
Balance
Housing & loan info
Campaign & contact details
Target variable: y (term deposit subscription)

## Key Steps in Analysis

Data Loading
The dataset is imported using pandas and inspected for structure and quality.

Exploratory Data Analysis (EDA)
Includes:
Age distribution
Job category distribution
Relationship between balance and subscription
Contact frequency trends
Subscription percentage breakdown

Visualizations
The notebook generates plots such as:
Histograms
Count plots
Pie charts
Bar charts
Boxplots
These visuals provide insights into customer demographics, financial behavior, and marketing campaign performance.

###Insights Highlight

Although insights may vary, typical findings include:
Middle-aged customers show higher subscription likelihood
Customers with stable jobs tend to subscribe more
Lower balances may correlate with lower subscription rates
Contact frequency has a direct impact on campaign success
